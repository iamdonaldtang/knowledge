# Event

Collaborative events involve joint promotions between multiple Quests within the same project or across different projects, making it easier to achieve strong promotional outcomes. TaskOn also periodically hosts official _Event_ activities.

<figure><img src="https://lh7-rt.googleusercontent.com/docsz/AD_4nXdVVp7MIYvittvz6oECV3g87s2Y7e5MBbYHCFaYSG8WxvEW_QBX-bNB8yS3vTzRB7cpy7GAvfIY00VzXmNUfovzP96Ey0pZRir5DFTxYgOsa0kzqmFaAj-BJxs24-3MMAJFM_d3?key=0Ugs8HwP_vZfQwQiqIZ9vA" alt=""><figcaption></figcaption></figure>

For setting up _Event_ activities, it’s recommended that project teams coordinate event details in advance. Each team can publish their own Quests, then jointly launch an Event online to expand the event’s scale and reward pool.

The first half of the Event setup process mirrors that of a Quest. Start by setting up the Quest, then create the event by entering basic information:  Event name, Event Period, and Event Image, followed by configuring the reward content.

<figure><img src="https://lh7-rt.googleusercontent.com/docsz/AD_4nXcROrijOWD5cyEyWN0uYiVQoEEk0udkoZUzCT3TFkcg8Tv-ff7BgF6nVAtnEYyMdtgSJcY7zUcLqaZE-q5PKrQ_jQNB-gKWGDanyKciawXilCiPmELOpydGCew3l4vZ2glteDebuQ?key=0Ugs8HwP_vZfQwQiqIZ9vA" alt=""><figcaption></figcaption></figure>

Next, select the tasks to include, choosing multiple tasks from your own project.

<figure><img src="https://lh7-rt.googleusercontent.com/docsz/AD_4nXejZqvMlV3RqUS9R2Hj-SsyhBxdEGFwUq16LlyqMsXd1_AACV0BxQk0u1sAEH6XxnIVPkDsN9HAKfKa59bwT6S8Ec1NkprChfMNo70yIs-dNRCN_77w1dE75JOA7_6UvWTGKYx9cQ?key=0Ugs8HwP_vZfQwQiqIZ9vA" alt=""><figcaption></figcaption></figure>

Then, Invite Co-Hosts. Once the Co-Host accepts the invitation, confirm and publish the Event.

## Event Referral

Project teams can set up event referral tasks and configure reward content to encourage users to promote the event organically.

Click _Referral_, then select _Create Referral_. Click _Next_.

<figure><img src="https://lh7-rt.googleusercontent.com/docsz/AD_4nXdKvY5ZS0HF8Tq0AO7zncgu0vYY_zlkdLw9PdHp9nTNhrXunMy9HZw9qy1jf3fhWtRs2emBrd7HcjG0Ay5Eg_3NT2OsQcrOEbLTx6Eg8sn031grWCA5j2ux7OWGjYYhJdGLVdbi?key=0Ugs8HwP_vZfQwQiqIZ9vA" alt=""><figcaption></figcaption></figure>

Choose the reward content, set the reward rules, and click _Publish_ to successfully launch the referral. Generous rewards can help the event reach and engage more participants.
