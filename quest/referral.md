# Referral

Project teams can set up Referral tasks by configuring basic information and referral reward content. Inviters who bring new users to join a Quest will earn rewards, making this a tool for driving viral campaign growth.

Once a Quest is created, the referral system allows viral promoters to earn rewards, with greater rewards for inviting more users.

<figure><img src="https://lh7-rt.googleusercontent.com/docsz/AD_4nXeNVsKMD8HjIH01y3MgAwHKwBKGYeoCXP5i8CbrMYyaNL5h7hQ5RNzIbw__rJJGdDZQEEJscOqu8K_vClISFglk8scISNKa59r90ZTbOYv1M9Ya8Ebp59N8Mgr7AD1hpdc-hgbFlA?key=0Ugs8HwP_vZfQwQiqIZ9vA" alt=""><figcaption></figcaption></figure>

By setting up a Referral system, you can maximize the Quest’s influence. Through referral rules, you can reward users who expand the campaign’s reach and impact.

## Usage Process

Click “Referral,” then select “Create Referral.”

Enter the referral name, add the task/event link, and include a description.

Click “Next.”

### Select Reward Type

Rewards can include Token, NFT, Whitelist, Points, Discord Role, and more. When setting rewards, after selecting the reward content, you must configure the winner rules.

<figure><img src="https://lh7-rt.googleusercontent.com/docsz/AD_4nXd-hmPqA-sFZZxEHHCaLSdQ8Du7LATNUsfVMtf1HXQ-LqKmlTW2d1_8g_ISoEkI_8Vf_njbO1uZ8x-TsV2YIYCtosbtOecIqRhfvlCBg11oTScK4XU3xDN-7k3v_XCvJ_qZ8YPF?key=0Ugs8HwP_vZfQwQiqIZ9vA" alt=""><figcaption></figcaption></figure>

**Set Winner Rankings:**

* **Top Number**: Specify the exact ranking positions of members. For example, members ranked 1st to 5th on the task leaderboard will be designated as winners.
* **Top Percentage**: Specify the top percentile range of members’ rankings.&#x20;

For example, top 10% to 20% means members ranked between the top 10% and 20% on the task leaderboard are eligible for rewards.

<figure><img src="https://lh7-rt.googleusercontent.com/docsz/AD_4nXfr9UtXMeLp33CA7QXLDK8YnMfpXxnI5BjWaTow0XzlvfWDdefLQWXt59-hhHWpgOrO8lrT4kmGR4sJzDp0OUC6jIdykTht8hImYaLIwbZ19Y1NKMvtPFveXM04kNjnBV_TDB1j?key=0Ugs8HwP_vZfQwQiqIZ9vA" alt=""><figcaption></figcaption></figure>

Winner rankings can be set without a limit on the number of winners. After configuring the basic content, you must also select how to distribute the prizes and set the minimum number of referrals required.

<figure><img src="https://lh7-rt.googleusercontent.com/docsz/AD_4nXfIxKjp5CSxnN6K1RhccVUCUrTZIIPETwSv6QfdOvf3PL485lbl-PeM7HhR2zLDYq2Sh3Q_hjDkKtztRX3O1r6b9D_jDVWbVd29m-LG4EE-wfUfws5OL2I9_Pe4_al-BxKCNKNw?key=0Ugs8HwP_vZfQwQiqIZ9vA" alt=""><figcaption></figcaption></figure>

Additionally, you can establish a multi-tier reward system, segmenting rewards based on the number of referrals.

<figure><img src="https://lh7-rt.googleusercontent.com/docsz/AD_4nXeHrW6WNlO7gLkZFuiEtEgzu4WKRtsqyZqqPy7HFpHyht3wslxvK4LkMQkSDeQ98m2CC1Et7gVkkW0F7xTzAAWXfTzC6MhUdaZghPD8Uv_RU0ZV9d_pDqJyBq3HphBuhSHN_bJ64A?key=0Ugs8HwP_vZfQwQiqIZ9vA" alt=""><figcaption></figcaption></figure>

Click “Publish.”
