# Proof of humanity

POH identity verification helps effectively identify high-quality, real users.

POH verification includes three options:**`Exchange KYC Identity`、`BABT Holder`、`zkMe Citizen Identity NFT Holder`；**

**Exchange KYC Identity**:&#x20;

Verification through KYC from exchanges like `Binance`, `Gate`, `OKX`, `Bybit`, `KuCoin`, or `Coinbase`.

<figure><img src="https://lh7-rt.googleusercontent.com/docsz/AD_4nXcqF7ZJp1SOL1OovfhdZG2ch0pLfgndpLIIFUijBYWtmfZEl0iEKDQbA6l1YP2M8NWCLbt7CSppTmDuoAGEP1hwjPcjsK5EwZWb85TUfkdzPUw7G1qZQpl1-F-kJY12P3Tne1EZ8Q?key=0Ugs8HwP_vZfQwQiqIZ9vA" alt=""><figcaption></figcaption></figure>

**BABT Holder**:&#x20;

Valid identity verification for users holding BABT (Binance Account Bound Token).

**zkMe Citizen Identity NFT Holder**:&#x20;

Verification based on NFTs issued for identity authentication.
