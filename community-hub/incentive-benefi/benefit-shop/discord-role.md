# Discord Role

## Set Up

### Basic Info

1. Enter the Discord Server URL.
2. If you have not added TaskOn bot, please add first ([**How to Add TaskOn Bot**](https://taskoncommunitys-organization.gitbook.io/entity-hub-for-business-end/function-cooperation/discord-bot-setting)).
3. Select Role.

<figure><img src="../../../.gitbook/assets/image (91).png" alt=""><figcaption></figcaption></figure>

### Set Redeem Rate

Please fill in the number; i.e., how many points are needed to redeem one Discord Role.

<figure><img src="../../../.gitbook/assets/image (92).png" alt=""><figcaption></figcaption></figure>

### Quantity-limited (Optional)

If your NFT is not unlimited or you want to control the quantity of Discord Role redemptions, enter the maximum supply.

<figure><img src="../../../.gitbook/assets/image (93).png" alt=""><figcaption></figcaption></figure>

### Time-limited (Optional)

If you want to set this benefit as time-limited, please enter the date.

<figure><img src="../../../.gitbook/assets/image (94).png" alt=""><figcaption></figcaption></figure>

### Participation Requirement (Optional)

If you want only participants within a specific ranking range to qualify for this benefit, click "Participation Requirement", then select "Leaderboard" for the specific range of ranking.

<figure><img src="../../../.gitbook/assets/image (95).png" alt=""><figcaption></figcaption></figure>

Click "`Next`".

<figure><img src="../../../.gitbook/assets/image (96).png" alt=""><figcaption></figcaption></figure>

## Info

1. Enter your benefit `name`, load the `cover image`, and add the `description`.
2. Click "`Publish`".
