# Promote Code

## Set Up

### Basic Info

1. Select type of code
2. If you choose multiple promote code, please download template, fill out the form, and then upload.



<figure><img src="../../../.gitbook/assets/image (114).png" alt=""><figcaption></figcaption></figure>

<figure><img src="../../../.gitbook/assets/image (115).png" alt=""><figcaption></figcaption></figure>

### Set Redeem Rate

Please fill in the number; i.e., how many points are needed to redeem the code.

<figure><img src="../../../.gitbook/assets/image (116).png" alt=""><figcaption></figcaption></figure>

### Time-limited (Optional)

If you want to set this benefit as time-limited, please enter the date.

<figure><img src="../../../.gitbook/assets/image (117).png" alt=""><figcaption></figcaption></figure>

### Participation Requirement (Optional)

If you want only participants from a specific ranking range to qualify for this benefit, click "Participation Requirement", then select "Leaderboard" for the specific range of ranking.



<figure><img src="../../../.gitbook/assets/image (118).png" alt=""><figcaption></figcaption></figure>

Click "`Next`".

<figure><img src="../../../.gitbook/assets/image (119).png" alt=""><figcaption></figcaption></figure>

## Info

1. Enter your benefit `name`, load the `cover image`, and add the `description`.
2. Click "`Publish`".
