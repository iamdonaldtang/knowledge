# NFT

## Set Up

### Basic info

Select network, NFT name, and enter the number of total reward amount.

Please be aware that if you opt for "NFT" as your benefit, ensure you have enough NFT on your TaskOn Asset. If your balance falls short, please [**deposit funds**](/broken/pages/uG3xxXUKLvEUytaCSl7y) beforehand.
