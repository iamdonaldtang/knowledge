# Joint Event

#### Objective

To aggregate project stakeholders of varying sizes to ensure fairness (more money, more customers; more effort, more money).

#### Recommended Tools

• Using Event

#### Recommended Configuration

• Event

￮ Quests from different projects are aggregated together, and each project contributes a fixed amount to establish a new prize pool. This prize pool consists of welfare tasks, used to reward users who complete XX% of the tasks (e.g., completing 80% of the tasks in the event).
