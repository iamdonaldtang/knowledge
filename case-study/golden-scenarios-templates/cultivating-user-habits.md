# Cultivating User Habits

#### Target

• Get users to come every day, create muscle memory, and increase DAU (Daily Active Users).

#### Recommended Tools

• DayChain

#### Recommended Configuration

• DayChain

**Day 1: Understanding the Project (Connect)**

￮ Task : Follow on Twitter + Join Discord + Visit the official website + Read the documentation/watch the videos + Complete the simple quiz (3 questions)

￮ Reward : 0.2 U

**Day 2: Increase Knowledge (Learn)**

￮ Task : Visit the Visit website + Read another document/watch a video + Complete a challenging quiz (5 questions) + Send a one-sentence chat message in the group.

￮ Reward : 0.3 U

**Day 3: Initial Interaction (Connect)**

￮ Task : Sign up on the product website or link your wallet to the product page + send two sentences in the group chat.

￮ Reward : 0.4 U

**Day 4: First On-Chain Interaction**

￮ Task : Low-cost on-chain interactions, such as: small-amount Swap/Mint Free NFT + two sentences in a group chat.

￮ Reward : 0.5 U (increases based on gas fees) + DC Role/NFT

**Day 5: Explore**

￮ Task : Medium-difficulty on-chain task + two sentences in group chat

￮ Reward : 0.7 U (increase based on gas fees)

**Day 6: Emotional Connection**

￮ Task : Light on-chain task + X post your thoughts/suggestions about the project with #hastag + 2 sentences in group chat.

￮ Reward : 1 U  (Increase amount customized based on gas fees)

**Day 7: The Ultimate Challenge (The Boss)**

￮ Tasks : Heavy on-chain tasks/core conversion actions (e.g., Swap > 10U, Stake, Mint NFT).

￮ Reward : 1.5U (increase based on gas fees)

#### **Select Eligibility**

1\. Choose two conditions, at least one of which must be met.

2\. X recommends 10 followers, at least 1 month age account and a non-default background restriction on profiles.

3\. POH is recommended to be greater than 20 points.
